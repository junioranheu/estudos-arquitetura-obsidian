# estudos-arquitetura-obsidian

Este repositório reúne minha base de conhecimento pessoal construída com o Obsidian.

Aqui documento meus estudos, anotações, resumos, diagramas e referências sobre desenvolvimento de software e tecnologia.

## Principais temas

- Arquitetura de Software
- Engenharia de Software
- Programação Orientada a Objetos (POO)
- Domain-Driven Design (DDD)
- Event-Driven Architecture
- CQRS
- Padrões de Projeto
- Bancos de Dados
- DevOps

## Objetivo

Centralizar meu aprendizado em um único lugar, registrando conhecimentos adquiridos ao longo da carreira para consulta, revisão e evolução contínua.
